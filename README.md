Run a second core for HA

## Usage

Pull repository

```bash
docker pull smeagolworms4/ha_second_core
```
Run container:

```bash
docker run -ti smeagolworms4/ha_second_core
```

## Environment variables

```
ENV BASE_CONFIG_PATH=BASE_CONFIG_PATH=second-core
```
